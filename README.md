# Instagram Lens 🚀

### Shifting Instagram from Passive Scrolling to Intentional Usage

## 🧠 Problem
- Ads are excessive and interruptive
- Users experience addictive but low-value scrolling
- Feed feels cluttered and irrelevant

Dataset : https://docs.google.com/spreadsheets/d/1LR3hWgpb7226h_zxP0-s25TYKbQgtNrlPHlTQQt6BVQ/edit?usp=sharing


## 💡 Solution: Instagram Lens

Users choose their intent before browsing:

- 👥 **Focus** — Social (friends & connections)
- ⚡ **Spark** — Inspiration (themed content)
- 🔍 **Discovery** — Normal mixed discovery feed

The UI, feed, and ads dynamically adapt based on the selected mode.

---

## ✨ Key Features

- Intent-based feed system
- Entry intent selection (before scrolling and can set a default mode in setting)
- Mode-based UI transformation
- Non-intrusive, intent-aligned ads
- AI-powered discovery interface

---

## 📈 Impact

### For Users:
- More meaningful sessions
- Reduced frustration
- Higher satisfaction

### For Business:
- Higher ad relevance & CTR
- Better retention quality
- Premium ad targeting

Instagram Lens shifts the platform from capturing attention to understanding intention.
